Planet Explorer

Bem-vindo ao Sistema Planet Explorer - um aplicativo para explorar informações sobre planetas do nosso sistema solar.

Esta é uma página de fallback simples enquanto o frontend completo está em construção ou enfrentando problemas técnicos.

Acesso à API

A API está disponível em: http://localhost:3000/api

Credenciais de Acesso:

Username: newadmin

Password: password123

Rotas Disponíveis:

  • Login: POST /api/users/login
  • Registro: POST /api/users/register
  • Listar Planetas: GET /api/planets

Para acessar a API, use o token JWT retornado após o login no cabeçalho de autorização: Authorization: Bearer <token>
